Technologies and Languages Popularity 2013

Consider a site that shows the day-to-day changes in the price of certain stocks for NASDAQ companies. Considering that StackOverflow keeps track of the number of questions per technology per day, I've figured we might be able to conduct the same kind of visual analysis to identify which technology are changing, and more importantly, how they are changing.

This uses archive.org to read the StackOverflow Tags page and its versions since the beginning of 2013 to calculate the percentage of change from the previous day.

Select Technologies to Compare

From this you should be able to select the languages you want, and with the bottom (smaller chart) select the timeframes you want to see.

If a technology cuts in the middle, it might be because it was no longer considered a "popular" tech by stack overflow.

 

Possible Caveats

The data from Archive.org isn't always accurate and some days are missing, therefore some drops (lower peaks) could be indicative of missing data.
