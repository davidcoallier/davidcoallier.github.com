Welcome to the #tcdisrupt finalist timeline

 

Select one of the finalist in the top menu to see the timeline of the last ~1000 tweets about them.

 

I am ashamed to be publishing this type of rough, unfinished work, however I only allowed myself to work on this for an hour this morning.

 

Each tweet is accompanied by a polarity, subjectivity and general positivity value. Please keep in mind that there are a few false positives, false negatives. For instance, floored, and ironically disrupt (amongst others) seem to be interpreted as negative.

 

How it works (or doesn't)

I'm using python-requests to iterate over 1000 last results for a search on "{name} AND %23tcdisrupt". When I get that, I analyse the polarity, subjectivity and positivity of each tweet using pattern.en, and build the story of each company using Timeline.js. I would have loved to see the "timeline-flags" colours according to the relative positivity but unfortunately, time dictated otherwise.
