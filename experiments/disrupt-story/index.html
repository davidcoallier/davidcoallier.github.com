Welcome to the #tcdisrupt finalist timeline

 

Select one of the finalist in the top menu to see the timeline of the last ~thousand tweets about them.

 

I am ashamed to be publishing this type of rough, unfinished work, however I only allowed myself to work on this for an hour this morning.

 

Each tweet is accompanied by a polarity, subjectivity and general positivity value. Please keep in mind that there are a few false positives, false negatives. For instance, floored, and ironically disrupt (amongst others) seem to be interpreted as negative.

 

How does it work (or doesn't)

I'm using python-requests to iterate over the last thousand search results for each company (e.g.: "{name} AND %23tcdisrupt"). I then proceed to analyse the polarity, subjectivity and positivity of each tweet using pattern.en, and build the story of each company using Timeline.js. I would have loved to see the "timeline-flags" coloured according to the relative positivity (red/yellow/greeN) but unfortunately, time dictated otherwise.
